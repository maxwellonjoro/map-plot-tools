# map-plot-tools
Simple project to plot points separated by a list of supported delimiters and draw them on a map. Points can be plotted as a path or as markers.

# tools
*   [LeafletJS](https://leafletjs.com/ "LeafletJS")
*   [AngularJS](https://angularjs.org/ "AngularJS")

**See it in action here: http://plot.maxonjoro.com/**
